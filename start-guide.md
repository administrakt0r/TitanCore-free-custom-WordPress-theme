# TitanCore WordPress Theme - Start Guide

Welcome to **TitanCore**, the highly optimized, modern, dark-mode ready WordPress theme!

## Quick Setup

1. **Install and Activate**
   Upload the exact folder into `wp-content/themes` or upload the `.zip` archive via the WordPress Admin panel (`Appearance > Themes > Add New`).

2. **Configure Menus**
   Head to `Appearance > Menus`.
   - **Primary Menu**: Used for your top navigation.
   - **Secondary Menu**: Used on the homepage to display categories or tags.
   - **Footer Menu**: Used at the bottom of the page.

3. **Homepage Display**
   Go to `Settings > Reading`. Ensure "Your homepage displays" is set strictly to "Your latest posts" if you want the default blog grid, OR set a static page and select the "Front Page" template if desired.

4. **Customizer Options**
   Go to `Appearance > Customize`.
   - Under **Header Options**, choose whether your header sticks to the top, and pick between showing a logo image or a styled text site title.
   - Under **Single Post Options**, you can toggle the automatic Table of Contents on or off.

5. **Performance**
   TitanCore is out-of-the-box optimized. Featured images, web fonts, and CSS are loaded with minimal blocking.

6. **Asset Files**
   TitanCore ships both source and minified runtime assets.
   - WordPress loads `assets/css/*.min.css` and `assets/js/navigation.min.js`.
   - If you edit `assets/css/*.css` or `assets/js/navigation.js`, update the matching minified files before release.
